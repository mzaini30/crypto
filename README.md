# Crypto Zen

Very simple encrypt and decrypt

## Installation

```bash
npm i crypto-zen
```

## Using

```typescript
import {encrypt, decrypt} from "crypto-zen"

console.log(encrypt(message, key))
console.log(decrypt(message, key))
```
